# indexs-uedsports-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可作为独立的展示、测试或工具页面使用，不针对任何特定域名或具体网站。

## 项目简介

`indexs-uedsports-lab` 是一个静态页面集合仓库，主要用于存放和发布各种独立 HTML 文件。每份文件均为完整的页面，可单独访问或嵌入其他项目。仓库内容以归档和实验性质为主，不包含动态后端或数据库依赖。

## 目录说明

```
indexs-uedsports-lab/
├── index.html          # 仓库入口页（可选）
├── pages/              # 主要页面归档目录
│   ├── page1.html      # 示例页面1
│   ├── page2.html      # 示例页面2
│   └── ...
└── assets/             # 资源文件（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立的 HTML 页面文件。
- `assets/`：存放页面引用的公共资源文件，按类型分类。

## 页面归档说明

- 每个 HTML 文件均为独立、自包含的页面，不依赖其他页面或外部服务。
- 页面之间无耦合关系，可单独部署或更新。
- 归档内容可能包含实验性、测试性或临时页面，不保证长期可用或兼容性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 若需引用仓库中的页面，请直接使用文件的原始链接（raw 版本）或通过 GitHub Pages 发布（需自行配置）。
- 不承诺对页面内容进行更新或技术支持，欢迎提交 Issue 或 Pull Request 进行贡献。

## 使用方式

1. 直接访问 `pages/` 目录下的 HTML 文件（通过 GitHub 文件浏览或 raw 链接）。
2. 将仓库克隆到本地，通过本地服务器（如 Live Server）查看效果。
3. 配置 GitHub Pages 后，可通过 `https://<用户名>.github.io/indexs-uedsports-lab/` 访问入口页面。
